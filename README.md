# n8n Custom Node Project

This repository contains a custom node for n8n, which is an open-source workflow automation tool.

## Installation

To install the custom node, follow these steps:
1. Clone the repository:
    ```bash
    git clone https://github.com/figphone/n8n_freepbx.git
    ```
2. Change directory into the project folder:
    ```bash
    cd n8n_freepbx
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

To use the custom node in n8n, follow these instructions:
1. Open n8n.
2. Navigate to the 'Nodes' section.
3. Select 'Add Node' and find your custom node.
